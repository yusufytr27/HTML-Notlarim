# 📚 HTML Ders Notları

Bu repo, **HTML öğrenme sürecimde tuttuğum ders notlarını ve örnek kodları** içermektedir.

## 📌 İçerik

Bu projede aşağıdaki HTML konularına ait örnekler ve notlar bulunmaktadır:

* HTML temel yapısı
* Başlık etiketleri
* Paragraf ve metin etiketleri
* Linkler
* Görseller
* Listeler
* Tablolar
* Formlar
* Semantic HTML

## 🎯 Amaç

Bu repository'nin amacı:

* HTML öğrenme sürecimi belgelemek
* Ders notlarını düzenli tutmak
* Başlangıç seviyesindeki kişilere örnekler sunmak

## 🚀 Kullanım

Projeyi bilgisayarınıza almak için:

```bash
git clone https://github.com/yusufytr27/HTML-Notlarim.git
```

Daha sonra HTML dosyalarını bir tarayıcıda açabilirsiniz.
